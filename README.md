# Bookmarklet: Copy URL Without Query Strings 
Copies the URL of the current tab without the URL parameters or query strings that often follow. You can extend the URL search params by hand if some of them are still there.

## How to Install Them
Installing a bookmarklet: 
Show your Bookmarks Bar: Press Ctrl + Shift + B (Windows/Linux) or Cmd + Shift + B (Mac) in Chrome, Firefox, and Edge. In Safari, go to View > Show Favorites Bar. 
Add a new bookmark: Create any bookmark and name it what you like. 
Insert the code: Replace the bookmark's destination URL (the "Location" or "URL" field) with the actual JavaScript code (which usually starts with javascript:).
Drag and Drop: Many sites provide a button for the bookmarklet; you can simply drag that button directly onto your Bookmarks Bar.
